## Convert timestamp to datetime

Usage

```bash
10:26 $ dt.py 
usage: dt.py [-h] [-u] timestamp
dt.py: error: the following arguments are required: timestamp

10:26 $ dt.py 1590202948
2020-05-23 10:02:28

10:27 $ dt.py 1590202948 --utc
2020-05-23 03:02:28
```
